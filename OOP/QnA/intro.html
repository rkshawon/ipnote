Object-oriented programming Object-oriented programming (OOP) is a programming paradigm based on the concept of objects, which can contain properties as data and function as methods for logic.
How Object Oriented Programming is related With Real world
Why we need OOPs in Programming language? OOPs provides code reusability which reduce the duplication of code which leads to performance. Code can be changed anytime or requirement of application changed anytime so when you want to make changes in your application, OOPs makes it easier
Some of the disadvantages of object-oriented programming include: 1. Steep learning curve: It can take time to get used to it.
2. Larger program size: Object-oriented programs typically involve more lines of code than procedural programs.
